## PiJS.app

### Modern javascript for Rasberry Pi

PiJS.app is fullstack, isomorphic JavaScript for the Raspberry Pi.
It's also React Progressive Web App. It's also Open Source.

#### Modules

- [User Entity](https://listingslab.com/userentity)
