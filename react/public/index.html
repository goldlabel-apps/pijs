Pi JS

A React PWA and Secure SSL Node/Express server running on a Raspberry Pi 4

After 20 years in the software industry we’ve started to get involved with hardware. Specifically the Raspberry Pi.

he Raspberry Pi is a very low cost, credit-card sized computer. It is a capable little device that enables people of all ages to explore computing, and to learn how to program in languages like Python. It can also give more experienced developers a chance to do some mad cool stuff they wouldn’t normally do. We’ve turned our Pi into a Node server, connected to the interweb through an everyday broadband connection.
